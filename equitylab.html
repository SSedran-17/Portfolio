Equity Lab
December 2022 - June 2023
Team Members: Michael Gee and Stefano Esposto

For my Website Development course in high school, I worked in a team of 3 to create a website to educate elementary school students about STEM trailblazers. A large part of the reason women and other underrepresented groups don't go into STEM is because of the lack of role models. Equity Lab is an equitable educational tool for the classroom. Built for students in Grades 5-8, Equity Lab provides games and study tools for students to learn and engage with the accomplishments of women, LGBTQ+, and other underrepresented people in STEM.

The user interface and the client side of the web application were implemented using Vue. We developed the backend of the web application using Express.js and managed API Routes using the API platform Postman. The data about each STEM Trailblazer was stored in a MongoDB Database.
